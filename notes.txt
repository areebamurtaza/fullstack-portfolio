3. Styling Rules
Option 1 - Custom CSS: Use your own styles.css file.
Option 2 - Tailwind CSS: Use Tailwind CSS CDN inside the HTML.
<script src="https://cdn.jsdelivr.net/npm/@tailwindcss/browser@4"></script>

Strict Rules
No Bootstrap.
No React.
No Next.js.
No copied full template.
No drag-and-drop website builders.

Personal introduction, education/background, internship goal, learning timeline.
Quote or mission statement and contact/address style information.
Required tags: article, section, blockquote, cite, time, address, dl, dt, dd.
Example: My goal during this internship is to become confident in frontend structure, GitHub workflow, and Next.js full-stack development.
Projects Page - projects.html
Project cards, skills table, learning progress, code snippet section, FAQ.
Required tags: article, table, caption, thead, tbody, tr, th, td, progress, meter, details, summary, pre, code.
Example table columns: Skill | Level | Practice Task.
Contact/Application Form Page - contact.html
This is important. Create the HTML form and fill it with sample data before taking screenshot.
Use fieldset and legend to group fields.
Use label for every input.
Use placeholder, required, minlength/maxlength, and phone pattern where useful.
No backend required. The goal is correct HTML form structure and filled screenshot.
Bonus Page - roadmap.html
Week 1 to Week 12 roadmap.
Ordered list and timeline style layout.
Current week highlighted.
Required tags: ol, li, time, mark, section, article.
