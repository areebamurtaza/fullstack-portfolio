Frequently Asked Questions

Common questions about the portfolio and internship process.

What technologies are used in this portfolio?

The portfolio is built with semantic HTML5 and custom CSS. It uses CSS Grid, Flexbox, and CSS variables for responsive, maintainable styling. No frameworks or backend required—it's a pure static website.

How many HTML tags are used?

Over 25 meaningful HTML tags are used throughout the site, including semantic tags like <header>, <nav>, <main>, <section>, <article>, <aside>, <footer>, form elements, tables, lists, and more.

Is the form functional?

The form is prefilled with sample data for demonstration purposes. It includes proper HTML5 validation attributes (required, minlength, maxlength, pattern) and demonstrates correct form structure with fieldsets, legends, and labels.

Can I deploy this to GitHub Pages?

Yes! Since this is a pure static website with no backend, it can be deployed directly to GitHub Pages. Just push to your repository and enable Pages in your repo settings.
